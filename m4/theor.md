# Отчет по моделированию движения шара по наклонной плоскости

## Описание модели

Данный код решает уравнения движения шарика радиуса $R$ и массы $m$, который катится с проскальзыванием по наклонной плоскости под углом $\theta$ к горизонтали.

## Физические формулы

### 1. Кинематические величины
- Положение: $\vec{r} = (x, y)$
- Линейная скорость: $\vec{v} = (v_x, v_y)$
- Угловая скорость вращения: $\omega$
- Скорость центра масс: $v = \sqrt{v_x^2 + v_y^2}$

### 2. Силы и моменты
- Нормальная реакция: $N = mg\cos\theta$
- Сила тяжести: $F_g = mg\sin\theta$ (вдоль плоскости)
- Скорость проскальзывания: $v_{\text{slip}} = v_{\parallel} - \omega R$, где $v_{\parallel} = \vec{v} \cdot \vec{e}_s$
- Направляющий вектор вдоль плоскости: $\vec{e}_s = (\cos\theta, \sin\theta)$

### 3. Два режима движения

#### Режим 1: Качение без проскальзывания
При $|v_{\text{slip}}| < \text{tol}$ и $|F_s| \leq \mu N$:
$$
a = \frac{mg\sin\theta}{m + I/R^2}
$$
$$
\vec{a} = a \cdot \vec{e}_s
$$
$$
\alpha = \frac{a}{R}
$$

#### Режим 2: Качение с проскальзыванием
При наличии проскальзывания:
- Сила трения: $F_f = \mu N$
- Направление силы трения: противоположно направлению проскальзывания
- Линейное ускорение: $\vec{a} = -\frac{\text{sign}(v_{\text{slip}})F_f}{m}\vec{e}_s + g\sin\theta\vec{e}_s$
- Угловое ускорение: $\alpha = \frac{\text{sign}(v_{\text{slip}})F_f R}{I}$

### 4. Энергетические величины
- Кинетическая энергия поступательного движения: $K_{\text{trans}} = \frac{1}{2}mv^2$
- Кинетическая энергия вращательного движения: $K_{\text{rot}} = \frac{1}{2}I\omega^2$
- Полная механическая энергия: $K_{\text{total}} = K_{\text{trans}} + K_{\text{rot}}$

## Численные методы

1. **Интегрирование уравнений движения**: используется `solve_ivp` из SciPy для решения системы ОДУ
2. **Условия остановки**: при $v < \text{tol}$ и $\omega < \text{tol}$ движение прекращается
3. **Интерполяция для анимации**: линейная интерполяция положения между точками решения
4. **Визуализация**: Matplotlib с анимацией в реальном времени
